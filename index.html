🧠 Гайд по учебе

1. Как эффективно учиться

  • Pomodoro: 25 минут учебы + 5 минут отдыха.
  • Active Recall: проверяйте себя вопросами.
  • Spaced Repetition: повторяйте материал через интервалы.
  • Объясняйте тему своими словами.

2. Как использовать приложения

  • Anki — запоминание через карточки.
  • Notion / OneNote — ведение конспектов.
  • Todoist — планирование задач.
  • Forest — концентрация без телефона.

3. Как работать с образовательными сайтами

  • Сначала изучайте теорию.
  • Потом решайте задачи.
  • Используйте несколько источников.

4. Подготовка к экзаменам

  • Решайте много заданий.
  • Разбирайте ошибки.
  • Повторяйте темы по плану.

5. Организация времени

  • Составляйте список задач на день.
  • Разделяйте сложные задачи на маленькие.
  • Делайте регулярные перерывы.
